現在 3 個。実は配列の関数が String にも使えたりするが……。

## toLower

文字を小文字にする関数だ。  
日本ではあんまり使わんな……。海外だと使うのかしら。
toKATAKANA とかあったら面白そうだけど(面倒くさそう)。

```ts
R.toLower("XYZ"); //=> 'xyz'
```

## toUpper

まあ toUpper もあるよね。大文字にする関数。

```ts
R.toUpper("abc"); //=> 'ABC'
```

## trim

空白を削除してくれる関数。
まあ string に同じメソッドあるけど……。

```ts
R.trim("   xyz  "); //=> 'xyz'
```
